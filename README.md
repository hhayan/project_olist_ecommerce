### RFM 기준
# 참고내용
- 금액: 저가 상품 중심 (화장품, 생활용품, 패션 액세서리)
- 대부분의 데이터가 왼쪽 하단에 밀집 (가격 0-500, 배송비 0-200)
- 가격 1000 이상은 극소수
- 저가 상품 중심 이커머스

이커머스 특성 반영
- 구매 주기 짧음
- 재구매 기회 많음 (다양한 카테고리)
* 아마존/이커머스 기준: 일반 고객의 재구매 주기가 평균 30~60일 수준임
(소비재·패션 중심: 약 30일, 가전/고가품은 90~120일 이상) *

이번 데이터에서 Frequency 분포를 보면:
1회 구매가 75% 이상
2회·3회 구매는 소수
4회 이상은 더더욱 희귀

# R (Recency) - 이커머스는 짧게!
5점: 0-30일 (최근) (즉시 리텐션/업셀 가능)
4점: 31-60일 (보통) 재구매 유도 타이밍
3점: 61-120일 (오래됌) 리마인드 프로모션 적기
2점: 121-180일 (이탈 위험) 이탈 조짐, 쿠폰 유도 필요
1점: 181일+ (휴면) 휴면 고객 대상 복귀 캠페인

# F(Frequency)
1점: 1회 구매
전체 고객: 94,763명 1회 구매(100%)

# M (Monetary)
5점: 500 이상 (고액)
4점: 300-499 (중고액)
3점: 150-299 (중액)
2점: 80-149 (중저액)
1점: 0-79 (소액)
